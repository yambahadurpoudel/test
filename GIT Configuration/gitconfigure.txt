4. To connect to the local
$git remote add origin "remote url"

5. To pull the branch
 $git pull origin master

6. to get staus
$git status

7. to remove file from staging area

$git rm --cached index.html


8. To discard the changes in working direcory
$git checkout <filename>

9. Adding git to local 
$git add .

10. To commit 
$git commit -m "any message"

11. Push to the remote
$git push origin master

12. to get commit no

$git log


13. to make branch
$git branch branch name
12. To change branch
$git checkout branch_name

14. Fetching all branch
$git fetch --all
